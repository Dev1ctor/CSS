Box-Model na prática

Vamos explorar um pouco do modelo de caixas, o unico modelo que não se encaixa no box-model é o inline-model e Links fazem parte deste modelo.


Como mostrado no exemplo acima, selecionando o h1 podemos ver na ferramenta do desenvolvedor(Para poder acessa-la pressione Cntrl+Shift+I / va nos três pontos no lado superior direito, clique em mais ferramentas e em ferramentas do desenvolvedor) ao lado que o h1 possui o height e width mas não possui padding e border e também possui margem.Isso se aplica a qualquer elemento que selecionarmos.


Border


Como mostra a imagem acima, configuramos algumas medidas para mostrar como exemplo(altura, largura da borda do h1) e essa ferramentado dev tools é de uso importante para para podermos encaixar certo como vamos querer que a largura e a posição que nossos elementos vão tomar na nossa página.Para isso basta clicar em cima de qualquer uma das configurações de estilo, e com a roda do mouse rolar para cima e para baixo para definir o tamanho que o elemento vai ocupar na tela e usando o dev tools não faz alterações no código original, facilitando para o desenvolvedor.


Padding

O padding fica responsável pelo preenchimento do espaço interno do border e pode ser alterado dentro de todos os lados assim como o border.


Margin

A margin é responsavel pelo que acontece em volta do caixa que o h1 ocupa, como mostrado na imagem podemos definir configurações para que ela mova a caixa do h1 tanto para a direita, quanto esquerda, baixo e cima, porém precisamos nos atentar a algo muito importante!


Importante Lembrar

Não podemos usar essas configurações da margin para centralizar nenhuma caixa,pois suponhamos que o navegador do cliente esteja sendo aberto no celular, pode ser que para ele esteja centralizado porém se ele abrir no Desktop a caixa ja não vai estar mais centralizada, para resolvermos isso vamos ter um pseudo-elementos específico que é o margin: auto; que ela vai centralizar nossa caixa horizontalmente de acordo com o tamanho da tela.


Podemos configurar também o outline da nossa caixa, ele é responsavel pelo tracejamento de uma linha em volta da nossa borda.

O box model é configurado da mesma maneira que o border, configurando somente a largura e o estilo desenhando uma linha do lado de dentro da margem como mostra a imagem acima.
